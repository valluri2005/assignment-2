# My College Event Web Page

## 📌 Description
This project is a basic HTML web page designed to showcase a college event. It demonstrates the use of HTML and CSS3 for styling and layout.

## 🎯 Objectives
- Apply inline, internal, and external CSS
- Understand CSS precedence
- Use different CSS selectors (element, class, ID)
- Style text using font and alignment properties
- Create and style lists
- Apply background styling using CSS3

## 🧩 Features
- Styled headings and paragraphs
- Event details section
- Highlighted content using class selector
- Event schedule list with improved readability
- Section with background color

## 🎨 CSS Used
1. Inline CSS → Applied directly inside HTML tags
2. Internal CSS → Defined inside <style> tag
3. External CSS → Linked via styles.css file

## ⚙️ How to Run
1. Save the HTML file as `index.html`
2. Save the CSS file as `styles.css`
3. Keep both files in the same folder
4. Open `index.html` in any web browser

## 📚 Concepts Covered
- HTML structure
- CSS selectors
- CSS precedence
- Text styling
- Background styling
- List styling

## ✅ Output
A simple and visually enhanced web page titled "My College Event"
